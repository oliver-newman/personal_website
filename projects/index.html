Projects

TeamThrive

TeamThrive (formerly Harvest Circle) is a project led by fellow Swarthmore College student Tyler Huntington and funded by Swarthmore's Lang Center for Civic & Social Responsibility. The mission of TeamThrive is to raise money to provide free dinners to high school students in Chester, PA, using a virtual charity run system. This involves a web-based platform built on Ruby on Rails and using the Strava API, which will allow users to track miles and earn money for the project pledged by corporate sponsors. The development version of the application is currently privately deployed on Heroku. Please contact me via email if you wish to have access to the application.

Homebrew

I am a contributor to the Homebrew-Core open-source project, a collection of package configuration files written in Ruby for Homebrew, the MacOS package manager. My main contribution thus far has been fixing system calls for 25 different packages to make them more robust.

GPA Calculator

The SCCS GPA Calculator is a tool for Swarthmore College students, for whom no other resource exists with which to easily calculate their GPAs. This calculator, however, has a twist; for a demonstration, click the "Fill with Sample" button on the calculator homepage. It was built using Python's Flask framework and Sympy math library, as well as a bit of Javascript and AJAX.
