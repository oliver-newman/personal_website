TeamThrive

TeamThrive (formerly Harvest Circle) is a project led by fellow Swarthmore College student Tyler Huntington and funded by Swarthmore's Lang Center for Civic & Social Responsibility. The mission of TeamThrive is to raise money to provide free dinners to high school students in Chester, PA, using a virtual charity run system. This involves a web-based platform built on Ruby on Rails and using the Strava API, which will allow users to track miles and earn money for the project pledged by corporate sponsors. The development version of the application is currently privately deployed on Heroku. Please contact me via email if you wish to have access to the application.
