Projects

TeamThrive

I am working with fellow Swarthmore student Tyler Huntington on TeamThrive (formerly Harvest Circle), a project funded by Swarthmore's Lang Center for Civic & Social Responsibility. The mission of TeamThrive is to raise money to provide free dinners to high school students in Chester, PA, using a "virtual charity run" system. This will involve a web-based platform built on Ruby on Rails, which will allow users to track miles and earn money for the project pledged by corporate sponsors.

Homebrew

I am a contributor to the Homebrew-Core open-source project, a collection of package configuration files written in Ruby for Homebrew, the MacOS package manager. My main contribution thus far has been fixing system calls for 25 different packages to make them more robust.
