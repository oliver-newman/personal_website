Projects

I am working with fellow Swarthmore student Tyler Huntington on Thrive (formerly Harvest Circle), a project funded by Swarthmore's Lang Center for Civic & Social Responsibility. The aim of Thrive is to raise money to provide free dinners to high school students in Chester, PA, using a "virtual charity run" system. This will involve a web-based platform built on Ruby on Rails, which will allow users to track miles and earn money for the project pledged by corporate sponsors.

I also redesigned (and am currently improving) the Swarthmore College Computer Society homepage using Jekyll; the development version can be viewed here.

Also check out Bubbles, a fun but completely useless interactive toy using p5.js.
